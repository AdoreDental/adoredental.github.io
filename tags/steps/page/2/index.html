    NewIssues

    jiraNewIssues This step creates new issues in bulk in the provided JIRA site. Input issues - issues to be created. site - Optional, default: JIRA_SITE environment variable. failOnError - Optional. default: true. auditLog - Optional. default: true. Append a panel to the comment with the build url and build user name. It supports all the fields that any jira instance supports including custom fields. For more information about all available input fields, please refer to the JIRA API documentation depending on your JIRA version. »

    Naresh Rayapati 12/11/2017 steps issue
